# LifeMate AI Demo Walkthrough

## Demo Goal

Show that LifeMate AI is more than a chatbot. It completes real life tasks through voice, local language, AI tools, offline mode, and family support.

## Recommended Demo Length

3 to 5 minutes.

## Demo Setup

User persona:

Name: Asha

Profile:

- 62 years old
- Prefers Hindi
- Has diabetes medicine reminders
- Daughter is connected through family dashboard
- Lives in a semi-urban/rural area
- Uses low-bandwidth mode

Family member:

Name: Priya

Role:

- Daughter/caregiver
- Receives missed medicine and SOS alerts

## Scene 1: Voice Assistant

Screen:

Home dashboard.

Presenter says:

"Asha does not need to type a perfect prompt. She just speaks in her own language."

User voice:

```text
LifeMate, meri diabetes ki dawa ka reminder roz subah nashta ke baad laga do.
```

Expected LifeMate response:

```text
Theek hai. Dawa ka naam bata dijiye, phir main roz nashta ke baad reminder laga dunga.
```

User:

```text
Metformin.
```

LifeMate:

```text
Main Metformin ka reminder roz subah nashta ke baad laga raha hoon. Kya aap chahengi ki agar reminder miss ho jaye to Priya ko alert mile?
```

User:

```text
Haan.
```

Result:

- Reminder created
- Caregiver alert enabled
- Dashboard updates

## Scene 2: Government Scheme Guidance

User:

```text
Kya mere liye koi health government scheme hai?
```

LifeMate:

```text
Main aapki madad karunga. Aap kis rajya mein rehti hain?
```

User:

```text
Madhya Pradesh.
```

LifeMate:

- Shows verified scheme cards
- Asks eligibility questions
- Lists required documents
- Offers save offline

Presenter says:

"For factual guidance, LifeMate uses verified knowledge retrieval instead of guessing."

## Scene 3: Document Summarizer and Translator

Action:

Upload a PDF or image of a scheme notice.

LifeMate output:

- 5-line simple summary
- Required documents
- Deadline
- Eligibility
- Translate button

Presenter says:

"A complex document becomes a simple action checklist in the user's language."

## Scene 4: Budget Planner

User:

```text
Meri monthly income 20000 hai. Mujhe dawa aur ghar ka kharcha manage karna hai.
```

LifeMate output:

- Basic monthly budget
- Medicine category
- Food/rent category
- Savings suggestion
- No risky financial advice

## Scene 5: Mental Wellness Check-In

User:

```text
Aaj thoda akela lag raha hai.
```

LifeMate:

- Responds warmly
- Suggests a short breathing exercise
- Offers to message family
- Does not diagnose

## Scene 6: Emergency SOS

Action:

Press and hold SOS button.

LifeMate:

- Starts countdown
- Shows location sharing
- Shows contacts to alert
- Sends mock alert to Priya

Family dashboard:

- Displays SOS event
- Location
- Medical snapshot
- Acknowledge button

Presenter says:

"The family dashboard is consent-based. It shares important safety signals, not private conversations."

## Scene 7: Offline Mode

Action:

Toggle network/offline simulation.

Show:

- Offline banner
- Medicine reminders still visible
- Saved scheme checklist available
- Pending sync queue

Presenter says:

"Even with weak internet, LifeMate keeps essential life tasks available."

## Winning Demo Ending

End with:

"LifeMate AI helps Asha remember medicine, understand benefits, manage money, get emotional support, and alert family in an emergency. It does this in her language, with safety and privacy built in."

## Backup Demo Plan

If live APIs fail:

- Use pre-recorded voice transcript
- Use mocked AI responses
- Show UI state transitions
- Mention that provider adapter supports OpenAI and Gemini
- Emphasize architecture and safety

## Judge-Facing Metrics to Mention

- Reminder created in under 20 seconds
- Scheme checklist generated in local language
- Document summarized into action items
- SOS alert reaches family dashboard
- Offline mode preserves reminders
- User completes multiple life tasks in one flow
